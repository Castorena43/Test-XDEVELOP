## Getting Started

First, install the packages

```bash
npm install
```

Second, add the env file with the paramters:

```bash
API_BOOKS=
API_KEY_REQRES=
API_URL_REQRES=
API_POSTS=
```

Third, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
